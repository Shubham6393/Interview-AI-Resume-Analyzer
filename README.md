🚀 AI Interview Platform

An AI-powered mock interview application built using the MERN Stack that enables users to practice interviews in a realistic environment. The application leverages Google Gemini AI to generate intelligent interview questions based on the user's resume and provides detailed feedback after each interview session.

✨ Features
🔐 Secure user authentication with JWT
📄 Resume upload functionality
🤖 AI-generated interview questions using Google Gemini AI
📝 Personalized interview sessions
📊 AI-generated interview reports and feedback
📁 Interview history management
☁️ MongoDB database integration
📱 Responsive and modern user interface
